# CLAUDE.md

## プロジェクト概要

Azure API Management (APIM) 経由で Azure OpenAI Service に接続する Python ベースの AI エージェントアプリケーション。
OpenAI Agents SDK を使用し、function calling による4つのツール（Excel読み込み、ファイル読み込み、Excel書き込み、ディレクトリ一覧）を備えた汎用アシスタント。
静的解析結果のトリアージ（誤検知/逸脱/修正の分類）にも対応。

## 技術スタック

- Python 3.13
- OpenAI Agents SDK (`openai-agents`)
- Azure OpenAI Service + Azure API Management
- python-dotenv（環境変数管理）
- openpyxl（Excel読み書き）

## プロジェクト構成

```
src/
├── config.py   # 環境変数の読み込み (.env)
├── client.py   # Azure OpenAI クライアントファクトリ
├── tools.py    # エージェントツール定義 (Excel読み書き/ファイル読み込み/ディレクトリ一覧)
├── agent.py    # エージェント構築 (システムプロンプト + ツール + ModelSettings)
└── main.py     # エントリポイント (CLI + 対話モード + 会話履歴管理)
```

## 開発コマンド

```bash
# セットアップ
python -m venv .venv
.venv/Scripts/activate        # Windows
# source .venv/bin/activate   # macOS/Linux
pip install -r requirements.txt
cp .env.example .env          # サブスクリプションキーを設定

# 実行（対話モード）
python -m src.main

# 実行（ワンショットモード）
python -m src.main "C:\results.xlsx を読み込んで内容を教えて"

# 構文チェック
python -m py_compile src/tools.py
python -m py_compile src/agent.py
python -m py_compile src/main.py
```

## 環境変数

`.env.example` を `.env` にコピーして以下を設定する：

| 変数名 | 必須 | 説明 |
|---|---|---|
| `AZURE_APIM_ENDPOINT` | Yes | Azure APIM のエンドポイント URL |
| `AZURE_APIM_SUBSCRIPTION_KEY` | Yes | APIM サブスクリプションキー |
| `AZURE_API_VERSION` | No | API バージョン（デフォルト: `2025-04-01-preview`） |
| `AZURE_DEPLOYMENT_NAME` | No | デプロイ名（デフォルト: `gpt-4o`） |
| `CLIENT_APPROACH` | No | クライアント方式: `azure`（推奨）または `openai` |

## ツール一覧

| ツール名 | 説明 |
|---|---|
| `read_excel` | Excelファイルの読み込み（ページネーション対応） |
| `read_source_code` | テキストファイルの読み込み（行番号指定、日本語エンコーディング自動判別） |
| `write_excel_cells` | Excelファイルへのセル書き込み（一括更新、列自動作成） |
| `list_directory` | ディレクトリ内容の一覧取得（globパターン対応） |

## コーディング規約

- ソースコード内のドキュメント文字列・UIテキストは日本語で記述する
- 非同期処理には `asyncio` を使用する
- ツール定義は `@function_tool` デコレータを使用する
- `.env` ファイルは Git にコミットしない（`.gitignore` に含まれている）
